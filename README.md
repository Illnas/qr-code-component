### Run Locally

Clone the project

```bash
  git clone https://github.com/Illnas/qr-code-component.git
```

Go to the project directory

```bash
  cd qr-code-component
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng start
```
